# 🔥 Windows

* Download the most recent .msi installer
* Double-click or right-click "Open" the installer
* Follow along the installation routine and finish the installation
* You will need administrative privileges to install the application

The applications folder can be found in `%programfiles%`.

🔥 The HCloud agent can be installed as a service on your Windows workstation. The application will then run in its own users context. Please take this into consideration when planning user privileges and access.&#x20;

_You should always consider the apps and the OS users privileges in your environment and set them according to your workflows needs and to internal security guidelines._
