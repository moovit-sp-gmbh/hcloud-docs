# Set a context for the HCloud agent

To use Hcloud, two things are needed:

* an account or token that is valid for your organization,
* a context in which you want to run the client (i.e. your organizations).

If you want to use your account to authorize the agent, make sure that you [enabled your account as a target](../../organizations/view-my-organizations/being-a-target.md) for the organization(s) you are working in.

Depending on the OS you are using, proceed with [MacOS and Windows](macos-and-windows.md) or [Docker (Linux)](docker-linux.md).
