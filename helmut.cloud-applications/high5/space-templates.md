# Space templates

## Creating spaces from a template

* In High5, go to the left side menu column and select "Spaces"
* Click "Create space" to start the wizard and choose to create a space out of a template
* Choose the template that you want to use
* Assign a name and click okay

Your space is ready now and already filled with the items from the template.



## Create a space template from an empty space

Create a space first as described in [Creating spaces from a template](https://app.gitbook.com/o/VfQ3Nav9YtSULN06QgZs/s/SBBPMUoB21gqFIYHfxuh/\~/changes/7/helmut.cloud-applications/high5/space-templates#creating-spaces-from-a-template). If you want to use it as a template, it makes sense to note this in the spaces name.

Modify and prepare the space as needed. If you are reusing certain event triggers and streams in a very similar way every time, you can prepare the repeating parts of a space up to a level of detail that suits your needs, and leave special configuration (webhooks, secrets and dedicated data) for later.
