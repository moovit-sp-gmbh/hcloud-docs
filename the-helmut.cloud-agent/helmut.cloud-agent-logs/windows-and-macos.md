# Windows and MacOS

To read the helmut.cloud Agents logfiles, do a right-click on the helmut.cloud Agent-icon and choose "_Logs_". You will be lead to the logs rightaway, called _"agent.log_".

With the log, you will be abled to view events that are executed on your machine, to supervise the connection to helmut.cloud and to see errors when they appear.

If you experience problems with helmut.cloud or some workflows and modules and you want to address support to solve them, please include the agent.log into your ticket, accompanied by a description of the problem and the time it appeared.
