# 🔥 View a spaces content

After creating a new space from scratch or with the help of a template, you can browse its properties and start to work with them. Click on a space in the space list to enter it.

A space contains:&#x20;

* Events&#x20;
* Streams connected to this events
* <mark style="background-color:orange;">A dashboard to monitor your spaces actions or Agents</mark>
* <mark style="background-color:orange;">A section to manage Agents which execute your streams</mark>
* Webhooks as an interface for incoming HTTP POST requests
* A Secrets Store to safely store passwords, variables or other secrets
* User Acces Control
* Space Settings
